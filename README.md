# 🔥 Flames | Awaken Your Plate

**The Elite Digital Landmark for Swat's #1 Fast Food Destination.**

This is a premium, high-conversion landing page designed for **Flames Cafe**, located in Continental Plazas, Mingora, Swat. It isn't just a website; it’s a cinematic digital experience built to drive hunger and capture high-intent customers.

---

## 💎 The Premium standard
Developed with the **Advanced Vibe Coding** framework, this project adheres to $100,000+ luxury website standards:

*   **Cinematic Visuals:** Custom-generated 3D assets for signature items (Zinger Burger, Crown Crust Pizza, Arabian Shawarma).
*   **Elite Typography:** A bold, multi-font system (`Bangers`, `Lilita One`, `Space Grotesk`) designed for impact and conversion.
*   **Buttery Interactions:** Powered by **GSAP (GreenSock)** and **Lenis.js** for inertia-driven, ultra-smooth scrolling and micro-animations.
*   **Social Proof Engine:** A robust, infinite-scrolling review module featuring real customer feedback from the Swat valley.
*   **"Wash" Navigation:** A professional drawer-based hamburger menu with staggered animations, perfectly optimized for mobile ergonomics.

---

## 🛠️ Technology Stack
- **Architecture:** Semantic HTML5, Mobile-First Design.
- **Styling:** Tailwind CSS (Utility-First, Production Optimized).
- **Animations:** GSAP (ScrollTrigger), CSS Keyframes.
- **Smooth Scroll:** Lenis.js (Inertia Scroll Engine).
- **Aesthetics:** Cinematic Noise Filter, Glassmorphism 2.0.
